# 快捷键说明
| 操作               | 说明     |
| :-------: | :-: |
| `Ctrl`+`S`         | 保存书源 |
| `Ctrl`+`Z`         | 撤销操作 |
| `Ctrl`+`Shift`+`Z` | 返回撤销 |
| `Ctrl`+`P` | 推送源 |
| `Shift`+`P` | 拉取源 |
| `Ctrl`+`E` | 生成源 |
| `Shift`+`C` | 清空表单 |
| `Shift`+`R`        | 重做操作 |
| `Ctrl`+`Enter`     | 调试源   |
