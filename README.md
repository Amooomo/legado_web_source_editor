# 🚀快捷键说明
> 移除`Shift`快捷键

| 操作               | 说明     |
| :-------- | :-: |
| `Ctrl`+`S`         | 保存书源 |
| `Ctrl`+`Z`         | 撤销操作 |
| `Ctrl`+`P` | 推送源 |
| `Ctrl`+`E` | 生成源 |
| `Ctrl`+`Enter`     | 调试源   |
